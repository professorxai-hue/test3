# TodoList RBAC

A React-based Todo List application with Role-Based Access Control (RBAC) authentication system.

## Features

- **User Authentication**: Secure login system with role-based access control
- **Todo Management**: Create, read, update, and delete todo items
- **Role-Based Permissions**: Different access levels based on user roles
- **Modern React Architecture**: Built with React 18 and React Router v6
- **Context API**: Centralized authentication state management using React Context
- **Component-Based Design**: Modular and reusable UI components

## Tech Stack

- React 18.2.0
- React Router DOM 6.x
- React Scripts 5.0.1
- Context API for state management

## Usage Examples

### Basic Usage

1. **Login**: Enter your credentials on the login page to access the application
2. **View Todos**: After authentication, you'll see your todo list
3. **Add Todo**: Use the input field to add new todo items
4. **Toggle Complete**: Click on a todo item to mark it as complete/incomplete
5. **Delete Todo**: Remove todos using the delete button

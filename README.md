# RBAC React App

A React application with Role-Based Access Control (RBAC) implementation.

## Features
- Role-based authentication (Admin, Editor, Viewer)
- Permission-based access control (Create, Read, Update, Delete)
- Protected routes
- Role gate components for conditional rendering

## Roles & Permissions
- **Admin**: Full access (Create, Read, Update, Delete)
- **Editor**: Create, Read, Update
- **Viewer**: Read only

## Usage
1. `npm install`
2. `npm start`
3. Login with any username and select a role
